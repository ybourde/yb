Research & Projects

Deep Generative Models for CT/PET Synthesis

Master's Thesis | Current
I am currently researching different ways to generate matching PET scans from existing CT scans using diffusion models. This project idea emerged from the fact that PET machinary is scarces relative to CT consequently reducing access to PET imagery to a large number of people. The goal is to see if existing diffusion models, particulary from stable diffusion, can be leveraged for this type of data, and if so how well. I am also exploring different training structures such as using different timesteps for each modalities.
  • Fine-tuning stable diffusion models on medical image data.
  • Exploring pre-processing techniques to handle extreme data sparsity of PET scans.
  • Exploring multi-timestep joint probability variant for medical image generation.
Tools: Python, PyTorch, HuggingFace.

Improving Volume Prediction of Wheat based on Multi-view Images

Semester Research Project | 2025 üìÑ Presentation üìë Report
For this research project, I picked up the work from a previous thesis and had to explore ways of improving the existing pipeline. The original idea was to be able to infer the volume of wheat spikes from 2D images taken from above. The pipeline was structured in 3 main components, a detection model, a segmentation model and then a volume prediction model. I did extensive work on the first 2 components by testing and fine-tuning different detection and segmentation models which could be improved. I explored a lot of different strategies to train the segmentation model with limited labeled data. Finally, I developed robust experiments to test the impact of improvements on the first 2 components on the final output volume.
  • Trained and benchmarked different object detection and segmentation models.
  • Explored and implemented different training strategies with limited labeled data.
  • Developed robust experiments to assess the impact of upstream models on final volume output.
Tools: Python, PyTorch, OpenCV.

Fine-tuning Foundational Model for EEG Classification

Semester Research Project | 2024 üìÑ Presentation
For this research project, my task was to use an in house foundation model trained on EEG data from a particular setting and see if it could be finetuned on different datasets which contained EEG data from other areas. The datasets that I chose to investigate were related to sleep analysis and image visualization classification (EEG data from looking at images). The second goal of the project was to try and reduce the model size such that it could fit on a small device. To this end, I used different model reduction techniques to reduce the model size while maintaining accuracy.
  • Finetuned of foundation model (LLM) for EEG data classification on 4 different datasets.
  • Developed and end-to-end pipeline from data to output classification.
  • Reduced model size by 50% using pruning techniques while maintaining accuracy.
Tools: Python, PyTorch, PyTorch Lightning, OmegaConf, TensorFlow Lite.
